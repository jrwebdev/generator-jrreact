# generator-jrreact

Yeoman generator for React web projects. Mostly just for personal use, given it uses a set list of
technologies based on personal preference:

* React
* Webpack
* Babel
* Sass
* Jest
* Enzyme

## Prerequisites

```
npm i -g yo
```

## Installation

```
npm i -g generator-jrrreact
```

## Usage
```
yo jrreact
```
